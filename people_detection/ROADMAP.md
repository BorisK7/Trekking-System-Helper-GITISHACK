# Дорожная карта развития проекта (Roadmap)

## 1. Оптимизация производительности
- [ ] **Асинхронная обработка:** Вынести чтение кадров, инференс модели и визуализацию в отдельные потоки/процессы.
- [ ] **TensorRT:** Конвертировать модели YOLOv8 в формат TensorRT для ускорения на NVIDIA GPU.
- [ ] **Batch Processing:** Реализовать пакетную обработку кадров для повышения пропускной способности.

## 2. Улучшение аналитики
- [ ] **Re-ID (Re-Identification):** Добавить модуль Re-ID (например, OSNet) для сохранения ID людей при выходе из кадра и возвращении (важно для multi-camera setup).
- [ ] **Траекторный анализ:** Кластеризация траекторий для выявления "популярных маршрутов" и аномального поведения.
- [ ] **Интеграция с базой данных:** Сохранение событий и метаданных в TimeScaleDB или PostgreSQL для исторического анализа.

## 3. Расширение функционала
- [ ] **Web Dashboard:** Создать веб-интерфейс (Streamlit/Flask) для просмотра статистики в реальном времени и управления настройками.
- [ ] **API Сервер:** Обернуть логику в REST API (FastAPI) для приема видеопотоков извне.
- [ ] **Поддержка RTSP:** Полноценная поддержка RTSP потоков с автоматическим переподключением.

## 4. Инфраструктура и качество кода
- [ ] **Docker:** Создать Dockerfile и docker-compose для простого развертывания (включая поддержку NVIDIA Runtime).
- [ ] **Тесты:** Написать Unit-тесты для модулей геометрии и аналитики.
- [ ] **CI/CD:** Настроить GitHub Actions для проверки кода (linter, tests).

## 5. Edge Computing
- [ ] **Jetson Optimizations:** Оптимизация под NVIDIA Jetson (Nano/Orin) с использованием DeepStream SDK.
